# Веб-приложение для обработки Excel-файлов с автомобильными дорогами

## Demo

Рабочая версия приложения доступна по ссылке:

https://irinai.pythonanywhere.com

## Описание проекта

Приложение разработано на **Django** и предназначено для обработки Excel-файлов с данными об автомобильных дорогах.

После загрузки файла приложение:

1. Проверяет структуру таблицы
2. Обрабатывает данные с использованием **Pandas**
3. Вычисляет общую протяжённость автомобильных дорог
4. Генерирует **Word-документ** с таблицей и аналитическим описанием
5. Предоставляет ссылку для скачивания готового отчета

Дополнительно реализована интеграция с **YandexGPT**, которая автоматически формирует краткое описание дорожной сети на основе загруженных данных.

# Функциональность приложения

## Загрузка Excel файла

Пользователь загружает Excel-файл через веб-интерфейс.

## Проверка структуры файла

Файл должен содержать следующие колонки **строго в указанном порядке**:

```
№ п/п
Наименование
Значение автомобильной дороги
Категория
Протяженность, км
```

Если структура не соответствует требованиям, пользователю выводится сообщение об ошибке.

---

## Обработка данных

После загрузки файл обрабатывается с помощью **Pandas**:

- чтение Excel
- очистка данных
- преобразование числовых значений
- вычисление суммарной протяженности дорог

---

## Генерация аналитического описания

На основе данных таблицы с помощью **YandexGPT** генерируется краткое описание дорожной сети.
Описание добавляется в отчет перед таблицей.

---

## Генерация Word-документа

С помощью библиотеки **python-docx** создаётся отчет со следующей структурой:

```
[Описание дорожной сети (сгенерировано AI)]

Таблица 1 – Перечень и характеристика автомобильных дорог,
проходящих по территории муниципального округа

[Таблица из Excel]

Общая протяженность автомобильных дорог составляет X км
```

# Установка и запуск проекта

## 1. Клонировать репозиторий

```bash
git clone https://github.com/Maslovalrina/road-reports-generator.git
cd road-reports-generator
```

## 2. Создать виртуальное окружение

```bash
python -m venv .venv
```

Активировать окружение:

Windows

```bash
.venv\Scripts\activate
```

Linux / MacOS

```bash
source .venv/bin/activate
```

## 3. Установить зависимости

```bash
pip install -r requirements.txt
```

## 4. Настроить переменные окружения для YandexGPT

В терминале задать:

```bash
YANDEX_CLOUD_API_KEY=your_api_key
YANDEX_CLOUD_FOLDER_ID=your_folder_id
```
your_api_key и your_folder_id необходимо заменить на соответствующие данные пользователя.
## 5. Применить миграции

```bash
python manage.py migrate
```

## 6. Запустить сервер

```bash
python manage.py runserver
```

Приложение будет доступно по адресу:

```
http://127.0.0.1:8000
```
